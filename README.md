<p align="center">
  <strong>$\LARGE \color{#FFC62D}{\textsf{DESCRIPTION}}$</strong>
</p>

<p align="center">
  <em>Optima is a modern event and outreach management platform powered by AI.</em><br/>
  <em>It helps teams plan smarter, collaborate seamlessly, and track real results — all in one place.</em><br/>
  <em>Built for organizations, student groups, or community teams who want to run events with clarity, structure, and measurable impact.</em>
</p>

<p align="center">
  <strong>$\LARGE \color{#FFC62D}{\textsf{CORE\ FEATURES}}$</strong>
</p>

<p align="center">
  • AI assistant (Jamie) for planning, reminders, and optimization<br/>
  • Voice-activated commands with wake-word detection and hands-free interaction<br/>
  • Event creation with goal tracking and real-time feedback<br/>
  • Role-based live collaboration for managing team members<br/>
  • Secure login and Firebase cloud sync across devices<br/>
  • AI Chat with memory, response filtering, and smart context<br/>
  • Fully customizable UI, themes, and in-app session settings<br/>
  • Interactive tutorials and a built-in support contact system
</p>

---

<p align="center">
  <strong>$\LARGE \color{#FFC62D}{\textsf{TECH\ STACK}}$</strong>
</p>

<p align="center">
  • <strong>Flutter 3+</strong> for cross-platform mobile UI<br/>
  • <strong>Firebase</strong>: Authentication, Firestore database, and Cloud Storage<br/>
  • <strong>Node.js + Express</strong> backend, deployed on <strong>Render</strong><br/>
  • <strong>OpenAI Whisper</strong> for voice-to-text transcription<br/>
  • <strong>GPT + TTS</strong> for assistant logic and voice responses<br/>
  • <strong>LiveKit</strong> for real-time audio and event communication<br/>
  • <strong>AdMob</strong> integration for rewarded ad monetization<br/>
  • <strong>Optima website</strong> built with <strong>HTML/CSS/JS</strong> and deployed via <a href="https://adisimaimulte1.github.io/optima-official-site/">GitHub Pages</a><br/>
  • <strong>Cron-job.org</strong> used for automatic backend pings and scheduled syncs
</p>

---

<p align="center">
  <strong>$\LARGE \color{#FFC62D}{\textsf{PLATFORM\ SUPPORT}}$</strong>
</p>

<p align="center">
  Note: Optima is available exclusively for <strong>mobile platforms</strong>. Desktop or web versions are not supported.<br/><br/>
  • <strong>Android:</strong> Build-ready using <code>flutter build apk</code> or <code>flutter run</code>. The latest version is also available on the <a href="https://adisimaimulte1.github.io/optima-official-site/">official site</a>.<br/>
  • <strong>iOS:</strong> Manual build required via <em>Xcode on macOS</em>. Developers were unable to get an iOS device to test the app.
</p>
